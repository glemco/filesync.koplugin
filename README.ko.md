# FileSync - KOReader용 무선 파일 관리자

[English](README.md) | [Español](README.es.md) | [Português](README.pt_BR.md) | [中文](README.zh_CN.md) | [العربية](README.ar.md) | [Français](README.fr.md) | [Deutsch](README.de.md) | [Русский](README.ru.md) | [日本語](README.ja.md) | **한국어**

전자책 리더에서 로컬 웹 서버를 실행하고 화면에 QR 코드를 표시하는 KOReader 플러그인입니다. 스마트폰으로 코드를 스캔하면 브라우저에서 깔끔한 웹 인터페이스가 열리며, 케이블이나 별도 앱 없이 책과 파일을 무선으로 관리할 수 있습니다.

KOReader가 설치된 **Kindle** 및 **Kobo** 기기에서 작동합니다.

## 기능

- **QR 코드 접속** — 스캔 한 번으로 즉시 연결, URL 직접 입력 불필요
- **파일 브라우저** — 경로 탐색(breadcrumb) 기능으로 라이브러리를 탐색
- **파일 업로드** — 드래그 앤 드롭 또는 탭으로 스마트폰에서 책을 업로드
- **파일 다운로드** — 한 번의 탭으로 원하는 파일을 스마트폰에 저장
- **폴더 생성** — 디렉터리를 만들어 라이브러리를 체계적으로 정리
- **이름 변경 및 삭제** — 확인 대화 상자가 포함된 기본적인 파일 관리
- **검색 및 정렬** — 이름으로 필터링, 이름/크기/날짜/유형별 정렬
- **다크 및 라이트 테마** — 자동 감지 또는 수동 전환
- **다양한 보기 모드** — 목록, 그리드, 대형 그리드 보기 지원
- **다국어 지원** — 10개 언어 지원 (영어, 스페인어, 포르투갈어, 중국어, 아랍어, 프랑스어, 독일어, 러시아어, 일본어, 한국어)
- **RTL 레이아웃 지원** — 아랍어를 위한 완전한 오른쪽에서 왼쪽 레이아웃 지원
- **슬립 방지** — 서버 실행 중 기기의 절전 모드 전환을 막고 WiFi 연결 유지
- **안전 모드** — 책과 이미지만 표시하고 시스템 파일은 숨김
- **반응형 UI** — 스마트폰에 최적화되어 있으며, 모든 화면 크기에서 작동

## 작동 방식

1. 전자책 리더를 WiFi에 연결합니다
2. KOReader의 네트워크 도구 메뉴에서 FileSync 플러그인을 엽니다
3. 전자책 리더 화면에 QR 코드가 나타납니다
4. 스마트폰으로 스캔합니다 (같은 WiFi에 연결되어 있어야 합니다)
5. 스마트폰 브라우저의 웹 인터페이스에서 책을 관리합니다

## 설치

### 사전 요구 사항

- [KOReader](https://github.com/koreader/koreader)가 설치된 Kindle 또는 Kobo 전자책 리더
- 전자책 리더와 스마트폰이 같은 WiFi 네트워크에 연결되어 있어야 합니다

### 방법 1: 릴리스 아카이브에서 설치 (권장)

1. [릴리스](../../releases) 페이지에서 최신 `.zip` 파일을 다운로드합니다
2. 아카이브를 압축 해제합니다
3. `filesync.koplugin` 폴더를 기기의 KOReader 플러그인 디렉터리에 복사합니다 (경로는 아래 참조)
4. KOReader를 재시작합니다

### 방법 2: 직접 복사

1. 전자책 리더를 USB로 컴퓨터에 연결합니다

2. KOReader 플러그인 디렉터리를 찾습니다:
   - **Kindle:** `/mnt/us/koreader/plugins/`
   - **Kobo:** `.adds/koreader/plugins/` (SD 카드 루트)

3. `filesync.koplugin` 폴더 전체를 플러그인 디렉터리에 복사합니다:
   ```
   plugins/
   ├── filesync.koplugin/
   │   ├── _meta.lua
   │   ├── main.lua
   │   └── filesync/
   │       ├── filesyncmanager.lua
   │       ├── httpserver.lua
   │       ├── fileops.lua
   │       ├── filesync_i18n.lua
   │       ├── json.lua
   │       ├── mobi.lua
   │       ├── utils.lua
   │       ├── static/
   │       │   └── index.html
   │       └── i18n/
   │           ├── en.po
   │           ├── es.po
   │           ├── pt_BR.po
   │           ├── zh_CN.po
   │           ├── ar.po
   │           ├── fr.po
   │           └── ...
   ├── other.koplugin/
   └── ...
   ```

4. 기기를 안전하게 분리하고 KOReader를 재시작합니다

### 설치 확인

KOReader를 재시작한 후 상단 메뉴를 열고 다음으로 이동합니다:

**네트워크 → FileSync**

메뉴 항목이 보이면 플러그인이 올바르게 설치된 것입니다.

## 사용법

### 서버 시작

0. 기기가 WiFi에 연결되어 있는지 확인합니다
1. KOReader 상단 메뉴를 엽니다
2. **네트워크 → FileSync**로 이동합니다
3. **Start file server**를 탭합니다
4. 화면에 접속 URL과 함께 QR 코드가 표시됩니다

### 스마트폰에서 연결

1. 스마트폰이 전자책 리더와 **같은 WiFi 네트워크**에 연결되어 있는지 확인합니다
2. 스마트폰 카메라를 열고 QR 코드를 스캔합니다
3. 링크를 탭하여 브라우저에서 웹 인터페이스를 엽니다
4. 또는 QR 코드 아래에 표시된 URL을 직접 입력합니다

### 파일 관리

연결되면 웹 인터페이스에서 다음 작업을 수행할 수 있습니다:

- **탐색** — 폴더를 탭하여 라이브러리를 탐색합니다. 상단의 경로 탐색 바를 사용하여 상위 디렉터리로 바로 이동할 수 있습니다.
- **업로드** — 헤더의 **Upload** 버튼을 탭한 후 파일을 선택하거나 드롭 영역에 드래그합니다. 여러 파일을 한 번에 업로드할 수 있습니다.
- **파일 상세** — 파일을 탭하면 상세 화면이 열리며, **다운로드**, **이름 변경**, **삭제**를 수행할 수 있습니다.
- **폴더 생성** — 헤더의 **Folder** 버튼을 탭하고 이름을 입력합니다.
- **검색** — 검색 바를 사용하여 현재 디렉터리를 파일 이름으로 필터링합니다.
- **정렬** — 드롭다운 메뉴를 사용하여 이름, 날짜, 크기, 유형별로 오름차순 또는 내림차순 정렬합니다.

### 슬립 방지

파일 서버가 실행 중인 동안 플러그인은 기기가 절전 모드로 전환되는 것을 자동으로 방지합니다. 이를 통해 서버 접근성과 WiFi 연결이 중단 없이 유지됩니다. 구체적으로:

- **대기 모드**와 **절전 모드** 전환이 차단되어 기기가 깨어 있는 상태를 유지합니다
- **자동 절전**과 **자동 대기** 타이머가 일시적으로 비활성화됩니다
- **WiFi 유지 기능**이 활성화되어 네트워크 연결이 유지됩니다

서버를 중지하면 모든 설정이 이전 값으로 복원됩니다. 기기가 어떤 이유로 절전 상태에 들어간 경우(배터리 잔량이 매우 부족한 경우 등), 기기가 깨어날 때 서버가 자동으로 재시작됩니다.

### 서버 중지

- 플러그인 메뉴에서 **Stop file server**를 탭하거나,
- KOReader를 종료하면 서버가 자동으로 중지됩니다

### 포트 변경

1. 플러그인 메뉴를 엽니다
2. **Server port**를 탭합니다
3. 1024에서 65535 사이의 포트 번호를 입력합니다 (기본값: 8080)
4. 변경 사항을 적용하려면 서버를 재시작합니다

### 안전 모드

안전 모드는 **기본적으로 활성화**되어 있으며, 웹 인터페이스에 독서 라이브러리와 관련된 파일만 표시합니다. 활성화 시:

- **전자책** (EPUB, PDF, MOBI, AZW3, FB2, DJVU, CBZ 등), **문서** (TXT, DOC, RTF, HTML 등), **이미지** (JPG, PNG, GIF, WebP)만 표시됩니다
- 시스템 파일, 설정 파일 및 기타 비도서 파일은 숨겨집니다
- KOReader 메타데이터 디렉터리(`.sdr` 폴더)는 숨겨지며, 책을 삭제할 때 자동으로 정리됩니다

안전 모드를 전환하려면 플러그인 메뉴를 열고 **Safe mode**를 탭하세요. 비활성화하면 기기의 모든 파일이 표시됩니다.

## 문제 해결

**플러그인이 메뉴에 표시되지 않는 경우**
- 폴더 이름이 정확히 `filesync.koplugin`인지 확인하세요 (대소문자 구분)
- `_meta.lua`와 `main.lua`가 폴더 바로 아래에 있는지 확인하세요 (중첩되지 않아야 합니다)
- KOReader를 완전히 재시작하세요

**"WiFi is not enabled" 오류**
- 서버를 시작하기 전에 전자책 리더를 WiFi 네트워크에 연결하세요
- 일부 기기에서는 KOReader의 네트워크 설정에서 WiFi를 명시적으로 활성화해야 합니다

**스마트폰에서 연결할 수 없는 경우**
- 두 기기가 같은 WiFi 네트워크에 연결되어 있는지 확인하세요
- QR 코드 스캔 대신 URL을 직접 입력해 보세요
- 라우터의 클라이언트 격리 기능이 활성화되어 있는지 확인하세요 (기기 간 통신을 차단함)
- Kindle의 경우: 플러그인이 방화벽 규칙을 자동으로 관리하지만, 규칙이 적용되지 않는 경우 재시작이 도움이 될 수 있습니다

**업로드가 실패하는 경우**
- 기기의 저장 공간을 확인하세요
- 매우 큰 파일은 시간 초과될 수 있습니다 — 작은 단위로 나누어 업로드해 보세요
- 대상 디렉터리에 쓰기 권한이 있는지 확인하세요
- 파일당 최대 업로드 크기는 1 GB입니다

**큰 파일 업로드 시 기기가 느려짐**
- 100 MB 이상의 파일을 업로드하면 전송 중 전자책 리더 UI가 일시적으로 응답하지 않을 수 있습니다. 이는 정상적인 현상입니다 — 기기의 처리 능력이 제한되어 있기 때문입니다. 업로드가 완료되면 UI가 정상으로 돌아옵니다.

## 기여하기

기여를 환영합니다!

1. 저장소를 포크합니다
2. 기능 브랜치를 생성합니다
3. 변경 사항을 적용합니다
4. 테스트 스위트를 실행합니다 (아래 참조)
5. 가능하면 실제 기기에서 테스트합니다
6. 풀 리퀘스트를 제출합니다

### 테스트 실행

이 프로젝트는 단위 테스트에 [busted](https://lunarmodules.github.io/busted/)를 사용합니다. 테스트는 순수 로직 함수(JSON 인코딩/디코딩, 경로 유효성 검사, 버전 파싱 등)를 다루며, KOReader 환경이 필요하지 않습니다.

**busted 설치** (아직 설치하지 않은 경우):

```bash
luarocks install busted
```

**모든 테스트 실행:**

```bash
busted
```

이 명령은 5개의 테스트 파일에 걸쳐 177개 이상의 테스트를 실행합니다. 다음과 같은 출력이 표시됩니다:

```
177 successes / 0 failures / 0 errors / 0 pending : 0.04 seconds
```

**특정 테스트 파일 실행:**

```bash
busted spec/json_spec.lua
```

**테스트 파일:**

| 파일 | 범위 |
|------|------|
| `spec/json_spec.lua` | JSON 인코딩/디코딩 왕복 테스트, 엣지 케이스, 오류 처리 |
| `spec/fileops_spec.lua` | 경로 탐색 방지, 파일명 유효성 검사, 크기 포맷팅, MIME 타입 |
| `spec/updater_spec.lua` | 버전 파싱, 버전 비교, 변경 로그 추출 |
| `spec/utils_spec.lua` | 플러그인 디렉터리 확인, 셸 이스케이프 |
| `spec/httpserver_spec.lua` | URL 디코딩, 쿼리 문자열 파싱 |

새로운 기능을 추가할 때는 순수 로직 함수에 대한 해당 테스트도 함께 추가해 주세요.
