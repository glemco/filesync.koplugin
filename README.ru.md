# FileSync - Беспроводной файловый менеджер для KOReader

[English](README.md) | [Español](README.es.md) | [Português](README.pt_BR.md) | [中文](README.zh_CN.md) | [العربية](README.ar.md) | [Français](README.fr.md) | [Deutsch](README.de.md) | **Русский** | [日本語](README.ja.md) | [한국어](README.ko.md)

Плагин для KOReader, который запускает локальный веб-сервер на вашей электронной книге и отображает QR-код на экране. Отсканируйте код телефоном — и в браузере откроется удобный веб-интерфейс для управления книгами и файлами без проводов. Никаких кабелей, никаких приложений — только ваш браузер.

Работает на устройствах **Kindle** и **Kobo** с установленным KOReader.

## Возможности

- **Доступ по QR-коду** — Сканируйте и подключайтесь мгновенно, без ввода URL
- **Файловый браузер** — Просматривайте библиотеку с навигацией по «хлебным крошкам»
- **Загрузка файлов** — Перетаскивайте или нажимайте для загрузки книг с телефона
- **Скачивание файлов** — Сохраняйте любой файл на телефон одним нажатием
- **Создание папок** — Организуйте библиотеку с помощью каталогов
- **Переименование и удаление** — Базовое управление файлами с диалогами подтверждения
- **Поиск и сортировка** — Фильтрация по имени, сортировка по имени/размеру/дате/типу
- **Тёмная и светлая темы** — Определяются автоматически или переключаются вручную
- **Несколько режимов отображения** — Список, сетка и крупная сетка
- **Многоязычная поддержка** — Доступен на 10 языках (английский, испанский, португальский, китайский, арабский, французский, немецкий, русский, японский, корейский)
- **Поддержка RTL** — Полная поддержка интерфейса справа налево для арабского языка
- **Защита от засыпания** — Устройство не уходит в сон, а WiFi остаётся активным, пока работает сервер
- **Безопасный режим** — Отображаются только книги и изображения, системные файлы скрыты
- **Адаптивный интерфейс** — Разработан для смартфонов, работает на любом экране

## Как это работает

1. Подключите электронную книгу к WiFi
2. Откройте плагин FileSync из меню «Сеть» в KOReader
3. На экране устройства появится QR-код
4. Отсканируйте его телефоном (подключённым к той же сети WiFi)
5. Управляйте книгами через веб-интерфейс в браузере телефона

## Установка

### Необходимые условия

- Электронная книга Kindle или Kobo с установленным [KOReader](https://github.com/koreader/koreader)
- Электронная книга и телефон должны быть подключены к одной сети WiFi

### Вариант 1: Из архива релиза (рекомендуется)

1. Скачайте последний релиз `.zip` со страницы [Релизы](../../releases)
2. Распакуйте архив
3. Скопируйте папку `filesync.koplugin` в каталог плагинов KOReader на вашем устройстве (см. пути ниже)
4. Перезапустите KOReader

### Вариант 2: Прямое копирование

1. Подключите электронную книгу к компьютеру через USB

2. Найдите каталог плагинов KOReader:
   - **Kindle:** `/mnt/us/koreader/plugins/`
   - **Kobo:** `.adds/koreader/plugins/` (в корне SD-карты)

3. Скопируйте всю папку `filesync.koplugin` в каталог плагинов:
   ```
   plugins/
   ├── filesync.koplugin/
   │   ├── _meta.lua
   │   ├── main.lua
   │   └── filesync/
   │       ├── filesyncmanager.lua
   │       ├── httpserver.lua
   │       ├── fileops.lua
   │       ├── filesync_i18n.lua
   │       ├── json.lua
   │       ├── mobi.lua
   │       ├── utils.lua
   │       ├── static/
   │       │   └── index.html
   │       └── i18n/
   │           ├── en.po
   │           ├── es.po
   │           ├── pt_BR.po
   │           ├── zh_CN.po
   │           ├── ar.po
   │           ├── fr.po
   │           └── ...
   ├── other.koplugin/
   └── ...
   ```

4. Безопасно извлеките устройство и перезапустите KOReader

### Проверка установки

После перезапуска KOReader откройте верхнее меню и перейдите:

**Сеть → FileSync**

Если вы видите этот пункт меню — плагин установлен корректно.

## Использование

### Запуск сервера

0. Убедитесь, что устройство подключено к WiFi
1. Откройте верхнее меню KOReader
2. Перейдите в **Сеть → FileSync**
3. Нажмите **Start file server** (Запустить файловый сервер)
4. На экране появится QR-код с URL для подключения

### Подключение с телефона

2. Откройте камеру телефона и отсканируйте QR-код
3. Нажмите на ссылку, чтобы открыть веб-интерфейс в браузере
4. Или введите URL, показанный под QR-кодом, вручную

### Управление файлами

После подключения веб-интерфейс позволяет:

- **Просмотр** — Нажимайте на папки для навигации по библиотеке. Используйте панель «хлебных крошек» вверху для перехода к любому родительскому каталогу.
- **Загрузка** — Нажмите кнопку **Upload** (Загрузить) в заголовке, затем выберите файлы или перетащите их в зону загрузки. Можно загружать несколько файлов одновременно.
- **Информация о файле** — Нажмите на любой файл, чтобы открыть его детальный вид, где можно **скачать**, **переименовать** или **удалить** файл.
- **Создание папок** — Нажмите кнопку **Folder** (Папка) в заголовке и введите имя.
- **Поиск** — Используйте строку поиска для фильтрации текущего каталога по имени файла.
- **Сортировка** — Используйте выпадающий список для сортировки по имени, дате, размеру или типу в прямом или обратном порядке.

### Защита от засыпания

Пока файловый сервер работает, плагин автоматически предотвращает переход устройства в спящий режим. Это обеспечивает постоянную доступность сервера и стабильное подключение WiFi. В частности:

- **Ожидание** и **засыпание** блокируются, чтобы устройство оставалось активным
- Таймеры **автоматического засыпания** и **автоматического ожидания** временно отключаются
- Включается **поддержание WiFi-соединения** для сохранения сетевого подключения

Все настройки возвращаются к прежним значениям при остановке сервера. Если устройство всё же перейдёт в спящий режим (например, из-за критически низкого заряда батареи), сервер автоматически перезапустится при пробуждении устройства.

### Остановка сервера

- Нажмите **Stop file server** (Остановить файловый сервер) в меню плагина, или
- Сервер остановится автоматически при выходе из KOReader

### Изменение порта

1. Откройте меню плагина
2. Нажмите **Server port** (Порт сервера)
3. Введите номер порта от 1024 до 65535 (по умолчанию: 8080)
4. Перезапустите сервер, чтобы изменения вступили в силу

### Безопасный режим

Безопасный режим **включён по умолчанию** и ограничивает веб-интерфейс отображением только файлов, относящихся к вашей библиотеке. При включении:

- Отображаются только **электронные книги** (EPUB, PDF, MOBI, AZW3, FB2, DJVU, CBZ и др.), **документы** (TXT, DOC, RTF, HTML и др.) и **изображения** (JPG, PNG, GIF, WebP)
- Системные файлы, файлы конфигурации и прочие файлы, не относящиеся к книгам, скрыты
- Каталоги метаданных KOReader (папки `.sdr`) скрыты и автоматически удаляются при удалении книги

Чтобы переключить безопасный режим, откройте меню плагина и нажмите **Safe mode** (Безопасный режим). Отключение покажет все файлы на устройстве.

## Устранение неполадок

**Плагин не отображается в меню**
- Убедитесь, что папка называется именно `filesync.koplugin` (с учётом регистра)
- Проверьте, что файлы `_meta.lua` и `main.lua` находятся непосредственно внутри папки (без вложенности)
- Полностью перезапустите KOReader

**Ошибка «WiFi is not enabled» (WiFi не включён)**
- Подключите электронную книгу к сети WiFi перед запуском сервера
- На некоторых устройствах необходимо явно включить WiFi в сетевых настройках KOReader

**Телефон не может подключиться**
- Убедитесь, что оба устройства находятся в одной сети WiFi
- Попробуйте ввести URL вручную вместо сканирования QR-кода
- Проверьте, не включена ли на роутере изоляция клиентов (она запрещает устройствам видеть друг друга)
- На Kindle: плагин управляет правилами файрвола автоматически, но перезагрузка может помочь, если правила «зависли»

**Ошибка при загрузке файлов**
- Проверьте свободное место на устройстве
- Очень большие файлы могут вызвать тайм-аут — попробуйте загружать файлы меньшими порциями
- Убедитесь, что целевой каталог доступен для записи
- Максимальный размер загрузки — 1 ГБ на файл

**Загрузка больших файлов замедляет устройство**
- Загрузка файлов размером более 100 МБ может привести к временной неотзывчивости интерфейса электронной книги во время передачи. Это нормально — устройство имеет ограниченную вычислительную мощность. Интерфейс восстановится после завершения загрузки.

### Запуск тестов

Проект использует [busted](https://lunarmodules.github.io/busted/) для модульного тестирования. Тесты покрывают функции чистой логики (кодирование/декодирование JSON, валидация путей, разбор версий и т.д.) и не требуют окружения KOReader.

**Установить busted** (если ещё не установлен):

```bash
luarocks install busted
```

**Запустить все тесты:**

```bash
busted
```

**Запустить конкретный файл тестов:**

```bash
busted spec/json_spec.lua
```

**Файлы тестов:**

| Файл | Покрытие |
|------|----------|
| `spec/json_spec.lua` | Кодирование/декодирование JSON, граничные случаи, обработка ошибок |
| `spec/fileops_spec.lua` | Защита от обхода путей, валидация имён файлов, форматирование размеров, MIME-типы |
| `spec/updater_spec.lua` | Разбор версий, сравнение версий, извлечение списка изменений |
| `spec/utils_spec.lua` | Определение каталога плагина, экранирование для shell |
| `spec/httpserver_spec.lua` | Декодирование URL, разбор строк запроса |

При добавлении новой функциональности, пожалуйста, добавляйте соответствующие тесты для функций чистой логики.
