# FileSync - KOReader用ワイヤレスファイルマネージャー

[English](README.md) | [Español](README.es.md) | [Português](README.pt_BR.md) | [中文](README.zh_CN.md) | [العربية](README.ar.md) | [Français](README.fr.md) | [Deutsch](README.de.md) | [Русский](README.ru.md) | **日本語** | [한국어](README.ko.md)

KOReaderプラグインで、電子書籍リーダー上にローカルウェブサーバーを起動し、画面にQRコードを表示します。スマートフォンでコードをスキャンすると、ブラウザ上で書籍やファイルをワイヤレスで管理できる洗練されたウェブインターフェースが開きます。ケーブルも専用アプリも不要 -- ブラウザだけで利用できます。

KOReaderがインストールされた**Kindle**および**Kobo**デバイスで動作します。

## 機能

- **QRコードアクセス** — スキャンするだけで即座に接続。URLの手入力は不要です
- **ファイルブラウザ** — パンくずナビゲーションでライブラリを閲覧できます
- **ファイルのアップロード** — ドラッグ＆ドロップまたはタップでスマートフォンから書籍をアップロードできます
- **ファイルのダウンロード** — ワンタップで任意のファイルをスマートフォンに保存できます
- **フォルダの作成** — ディレクトリを作成してライブラリを整理できます
- **名前変更と削除** — 確認ダイアログ付きの基本的なファイル管理機能です
- **検索とソート** — 名前でフィルタリング、名前・サイズ・日付・種類でソートできます
- **ダーク＆ライトテーマ** — 自動検出または手動で切り替えられます
- **複数の表示モード** — リスト、グリッド、大型グリッドの各ビューに対応しています
- **多言語対応** — 10言語で利用可能です（英語、スペイン語、ポルトガル語、中国語、アラビア語、フランス語、ドイツ語、ロシア語、日本語、韓国語）
- **RTLレイアウト対応** — アラビア語向けの完全な右から左のレイアウトに対応しています
- **スリープ防止** — サーバー稼働中はデバイスのスリープを防ぎ、WiFi接続を維持します
- **セーフモード** — 書籍と画像のみを表示し、システムファイルを非表示にします
- **レスポンシブUI** — スマートフォン向けに設計されていますが、どの画面サイズでも動作します

## 仕組み

1. 電子書籍リーダーをWiFiに接続します
2. KOReaderのネットワークツールメニューからFileSyncプラグインを開きます
3. 電子書籍リーダーの画面にQRコードが表示されます
4. スマートフォンでスキャンします（同じWiFiに接続してください）
5. スマートフォンのブラウザに表示されるウェブインターフェースで書籍を管理します

## インストール

### 前提条件

- [KOReader](https://github.com/koreader/koreader)がインストールされたKindleまたはKobo電子書籍リーダー
- 電子書籍リーダーとスマートフォンが同じWiFiネットワークに接続されていること

### 方法1: リリースアーカイブから（推奨）

1. [リリース](../../releases)ページから最新の`.zip`ファイルをダウンロードします
2. アーカイブを展開します
3. `filesync.koplugin`フォルダをデバイスのKOReaderプラグインディレクトリにコピーします（パスは下記を参照）
4. KOReaderを再起動します

### 方法2: 直接コピー

1. 電子書籍リーダーをUSBでコンピュータに接続します

2. KOReaderのプラグインディレクトリを見つけます:
   - **Kindle:** `/mnt/us/koreader/plugins/`
   - **Kobo:** `.adds/koreader/plugins/`（SDカードのルート）

3. `filesync.koplugin`フォルダ全体をプラグインディレクトリにコピーします:
   ```
   plugins/
   ├── filesync.koplugin/
   │   ├── _meta.lua
   │   ├── main.lua
   │   └── filesync/
   │       ├── filesyncmanager.lua
   │       ├── httpserver.lua
   │       ├── fileops.lua
   │       ├── filesync_i18n.lua
   │       ├── json.lua
   │       ├── mobi.lua
   │       ├── utils.lua
   │       ├── static/
   │       │   └── index.html
   │       └── i18n/
   │           ├── en.po
   │           ├── es.po
   │           ├── pt_BR.po
   │           ├── zh_CN.po
   │           ├── ar.po
   │           ├── fr.po
   │           └── ...
   ├── other.koplugin/
   └── ...
   ```

4. デバイスを安全に取り外し、KOReaderを再起動します

### インストールの確認

KOReaderの再起動後、上部メニューを開いて以下に移動します:

**ネットワーク → FileSync**

メニュー項目が表示されていれば、プラグインは正しくインストールされています。

## 使い方

### サーバーの起動

0. デバイスがWiFiに接続されていることを確認します
1. KOReaderの上部メニューを開きます
2. **ネットワーク → FileSync**に移動します
3. **Start file server**をタップします
4. 接続URLとともにQRコードが画面に表示されます

2. スマートフォンのカメラを開き、QRコードをスキャンします
3. リンクをタップしてブラウザでウェブインターフェースを開きます
4. または、QRコードの下に表示されているURLを手動で入力します

### ファイルの管理

接続すると、ウェブインターフェースで以下の操作ができます:

- **閲覧** — フォルダをタップしてライブラリ内を移動します。上部のパンくずバーを使って、任意の上位ディレクトリに直接移動できます。
- **アップロード** — ヘッダーの**Upload**ボタンをタップし、ファイルを選択するかドロップゾーンにドラッグします。複数ファイルを一度にアップロードできます。
- **ファイル詳細** — 任意のファイルをタップすると詳細画面が開き、**ダウンロード**、**名前変更**、**削除**の操作ができます。
- **フォルダの作成** — ヘッダーの**Folder**ボタンをタップし、名前を入力します。
- **検索** — 検索バーを使って、現在のディレクトリ内をファイル名でフィルタリングします。
- **ソート** — ドロップダウンメニューを使って、名前・日付・サイズ・種類で昇順または降順にソートします。

### スリープ防止

ファイルサーバーの稼働中、プラグインはデバイスがスリープやサスペンド状態に入るのを自動的に防止します。これにより、サーバーへのアクセスとWiFi接続が中断なく維持されます。具体的には:

- **スタンバイ**と**サスペンド**がブロックされ、デバイスが起動状態を維持します
- **自動サスペンド**と**自動スタンバイ**のタイマーが一時的に無効になります
- **WiFiキープアライブ**が有効になり、ネットワーク接続が維持されます

サーバーを停止すると、すべての設定は以前の値に復元されます。何らかの理由でデバイスがサスペンドした場合（バッテリー残量が極めて低い場合など）、デバイスの復帰時にサーバーが自動的に再起動します。

### サーバーの停止

- プラグインメニューから**Stop file server**をタップするか、
- KOReaderを終了すると、サーバーは自動的に停止します

### ポートの変更

1. プラグインメニューを開きます
2. **Server port**をタップします
3. 1024から65535の範囲でポート番号を入力します（デフォルト: 8080）
4. 変更を反映するためにサーバーを再起動します

### セーフモード

セーフモードは**デフォルトで有効**になっており、ウェブインターフェースに読書ライブラリに関連するファイルのみを表示します。有効時:

- **電子書籍**（EPUB、PDF、MOBI、AZW3、FB2、DJVU、CBZなど）、**ドキュメント**（TXT、DOC、RTF、HTMLなど）、**画像**（JPG、PNG、GIF、WebP）のみが表示されます
- システムファイル、設定ファイル、その他書籍以外のファイルは非表示になります
- KOReaderのメタデータディレクトリ（`.sdr`フォルダ）は非表示になり、書籍を削除する際に自動的にクリーンアップされます

セーフモードを切り替えるには、プラグインメニューを開いて**Safe mode**をタップしてください。無効にすると、デバイス上のすべてのファイルが表示されます。

## トラブルシューティング

**プラグインがメニューに表示されない**
- フォルダ名が正確に`filesync.koplugin`であることを確認してください（大文字小文字を区別します）
- `_meta.lua`と`main.lua`がフォルダの直下にあることを確認してください（入れ子になっていないこと）
- KOReaderを完全に再起動してください

**「WiFi is not enabled」エラー**
- サーバーを起動する前に電子書籍リーダーをWiFiネットワークに接続してください
- 一部のデバイスでは、KOReaderのネットワーク設定でWiFiを明示的に有効にする必要があります

**スマートフォンから接続できない**
- 両方のデバイスが同じWiFiネットワークに接続されていることを確認してください
- QRコードのスキャンの代わりにURLを手動で入力してみてください
- ルーターのクライアント分離が有効になっていないか確認してください（デバイス間の通信が制限されます）
- Kindleの場合: プラグインがファイアウォールルールを自動的に管理しますが、ルールが適用されない場合は再起動が有効です

**アップロードが失敗する**
- デバイスの空き容量を確認してください
- 非常に大きなファイルはタイムアウトする場合があります。小分けにしてアップロードしてみてください
- アップロード先のディレクトリに書き込み権限があることを確認してください

## コントリビューション

コントリビューションを歓迎いたします！

1. リポジトリをフォークします
2. フィーチャーブランチを作成します
3. 変更を加えます
4. テストスイートを実行します（下記参照）
5. 可能であれば実機でテストします
6. プルリクエストを送信します

### テストの実行

本プロジェクトでは、ユニットテストに [busted](https://lunarmodules.github.io/busted/) を使用しています。テストは純粋なロジック関数（JSONエンコード/デコード、パス検証、バージョン解析など）をカバーしており、KOReader環境は不要です。

**bustedのインストール**（未インストールの場合）：

```bash
luarocks install busted
```

**全テストの実行：**

```bash
busted
```

**特定のテストファイルを実行：**

```bash
busted spec/json_spec.lua
```

**テストファイル：**

| ファイル | カバー範囲 |
|----------|------------|
| `spec/json_spec.lua` | JSONエンコード/デコードの往復テスト、エッジケース、エラーハンドリング |
| `spec/fileops_spec.lua` | パストラバーサル防止、ファイル名検証、サイズフォーマット、MIMEタイプ |
| `spec/updater_spec.lua` | バージョン解析、バージョン比較、変更履歴の抽出 |
| `spec/utils_spec.lua` | プラグインディレクトリの解決、シェルエスケープ |
| `spec/httpserver_spec.lua` | URLデコード、クエリ文字列の解析 |

新機能を追加する際は、純粋なロジック関数に対応するテストも追加してください。
